# Blog Content

Read blog posts and content from Alex MacArthur's website.

## Usage

Blog posts are available at:
- Individual posts: `https://macarthur.me/posts/{slug}`
- Post listing: `https://macarthur.me/posts`
- Scraps (short posts): `https://macarthur.me/scraps/{slug}`

Request with `Accept: text/markdown` header to receive markdown versions of pages.
