# RSS Updates

Subscribe to updates from Alex MacArthur's website via RSS.

## Usage

The RSS feed is available at `https://macarthur.me/rss/feed.xml` in XML format.

The feed includes:
- Latest blog posts with titles, descriptions, and publication dates
- Links to full post content
- Content in both HTML and (when requested with `Accept: text/markdown`) markdown format
